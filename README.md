# Smart Child
 
Smart Child is an interactive web platform designed to enrich children's lives through fun, educational games and activities. Built with both kids and parents in mind, the platform features a dual-login system — children get a playful, age-appropriate experience, while parents get a dedicated dashboard to monitor progress, manage screen time, and stay involved in their child's learning journey.
 
## Key Features
 
- 🎮 Interactive games designed to boost cognitive and creative skills
- 👨‍👩‍👧 Separate parent & child login experiences
- 📊 Parent dashboard to track activity and progress
- 🔒 Safe, child-friendly environment
- 📱 Responsive design for all devices
